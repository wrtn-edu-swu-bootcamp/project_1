# 운동 플랜 자동 생성 서비스 기획안

## 📋 프로젝트 개요

### 서비스 명
**FitPlan AI** - 개인 맞춤형 운동 플랜 자동 생성 서비스

### 서비스 목표
사용자의 신체 정보, 운동 목표, 선호도를 바탕으로 AI가 자동으로 최적화된 운동 계획을 생성하고 관리하는 서비스

---

## 🎯 핵심 기능

### 1. 사용자 프로필 입력
- **신체 정보**
  - 키, 몸무게, 나이, 성별
  - 체지방률 (선택)
  - 현재 운동 수준 (초급/중급/고급)

- **운동 목표 설정**
  - 체중 감량
  - 근육 증가
  - 체력 향상
  - 건강 유지
  - 재활/통증 관리

- **운동 환경**
  - 운동 가능 장소 (홈트레이닝/헬스장)
  - 보유 장비
  - 주간 운동 가능 일수
  - 1회 운동 시간

### 2. AI 기반 운동 플랜 생성
- **자동 플랜 생성**
  - 사용자 정보 기반 최적 운동 루틴 생성
  - 주차별 진행 계획
  - 운동별 세트, 횟수, 무게 제안

- **운동 종류**
  - 웨이트 트레이닝
  - 유산소 운동
  - 스트레칭/유연성
  - 기능성 운동

### 3. 운동 기록 및 추적
- **일일 운동 기록**
  - 완료한 운동 체크
  - 실제 수행한 무게/횟수 기록
  - 운동 중 메모

- **진행 상황 모니터링**
  - 주간/월간 운동 통계
  - 목표 달성률
  - 신체 변화 기록 (체중, 사진)

### 4. 스마트 조정 기능
- **실시간 플랜 조정**
  - 수행 능력에 따른 난이도 자동 조정
  - 부상/피로도 고려한 플랜 변경
  - 정체기 돌파를 위한 플랜 수정

### 5. 운동 가이드 제공
- **운동 방법 안내**
  - 각 운동별 상세 설명
  - 영상/이미지 가이드
  - 올바른 자세 체크포인트

## 🛠 기술 스택

### Frontend
- React / Next.js
- TypeScript
- Tailwind CSS
- Chart.js (통계 시각화)

### Backend
- Node.js / Express
- Python (AI 모델)
- RESTful API

### Database
- MongoDB (사용자 데이터, 운동 기록)
- PostgreSQL (운동 데이터베이스)

### AI/ML
- OpenAI API (운동 플랜 생성)
- TensorFlow (사용자 패턴 분석)

### 인프라
- AWS / Vercel
- Docker
- CI/CD (GitHub Actions)

## 🎯 타겟 사용자

### Primary
- 20-40대 직장인
- 운동 초보자~중급자
- 헬스장 또는 홈트레이닝 수행자

### Secondary
- PT 비용 부담이 큰 사용자
- 체계적인 운동 관리를 원하는 사용자
- 자기 관리에 관심이 많은 사용자
